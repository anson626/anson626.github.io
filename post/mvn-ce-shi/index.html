2022-07-10 2 min read # Mvn

Install JDK

install the jdk version 8

yum -y install java-1.8.0-openjdk.x86_64 java-1.8.0-openjdk-devel.x86_64

------------------------------------------------

Install Maven

Maven In production we are 3.6.3 version

wget https://downloads.apache.org/maven/maven-3/3.6.3/binaries/apache-maven-3.6.3-bin.tar.gz -P /opt/
cd /opt/
tar xf /opt/apache-maven-3.6.3-bin.tar.gz
ln -s /opt/apache-maven-3.6.3 /opt/maven

vim /etc/profile.d/maven.sh

export M2_HOME=/opt/maven
export MAVEN_HOME=/opt/maven
export PATH=${M2_HOME}/bin:${PATH}

source /etc/profile.d/maven.sh

#Check version 
mvn --version

---------------------------------------------------------

Install Java Program - GUNS

yum install git -y

git clone https://gitee.com/stylefeng/guns.git

Running GUNS program need to Install mysql

vim /etc/yum.repos.d/MariaDB.repo

[mariadb]
name = MariaDB
baseurl = http://yum.mariadb.org/10.4/centos7-amd64
gpgkey=https://yum.mariadb.org/RPM-GPG-KEY-MariaDB
gpgcheck=1

yum -y install MariaDB-server MariaDB-client

#Check the installed package 
rpm -qi MariaDB-server

#Start and enable MariaDB service
systemctl start mariadb && systemctl enable mariadb

Provide user Privileges and import database

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '123456';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'localhost' IDENTIFIED BY '123456';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'127.0.0.1' IDENTIFIED BY '123456';

#must add guns tables
CREATE DATABASE guns;

#FYI the user password must same with the application 
/opt/guns/guns/src/main/resources/application-local.yml

In path pom.xml ，using maven compile and install

cd /opt/guns
mvn  clean  install -Dmaven.test.skip=true

After the installation is complete, you will see the jar package. Run the jar file

cd /opt/guns/target

java -jar  guns.jar # Run on front-end 
nohup java -jar  guns.jar &    #Run in back-end

Check the jar file status

jps -mlvV
